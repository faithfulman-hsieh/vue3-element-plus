# 界面定義：任務中心 (UI Task)

## 畫面描述
處理流程任務的核心工作台，包含三個主要頁籤。

## 頁籤與欄位 (Tabs & Fields)
- **待辦任務 (pending)**: 
    - `當前任務`, `流程名稱`, `收到時間`。
- **可認領任務 (group)**: 
    - `任務名稱`, `流程名稱`, `收到時間`。
- **經手任務 (history)**: 
    - `任務名稱`, `流程名稱`, `完成時間`, `當前處理人員`。

## 核心組件與行為
- **DynamicForm彈窗**: 
    - 根據 `getTaskForm` 返回的欄位動態渲染。
    - 點擊「確認完成」執行 `submitTaskForm`。
- **認領/取消認領**:
    - 在「可認領」與「待辦」清單間移動任務。
- **流程狀態檢視**:
    - 彈出介面使用 `BpmnViewer` 高亮顯示當前節點。
