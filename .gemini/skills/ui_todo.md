# 界面定義：簡易待辦 (UI Todo)

## 畫面描述
用於展示與操作基礎 CRUD 功能的簡易頁面。

## 欄位定義 (Fields)
- **新增表單**:
    - `title` (字串): 待辦標題。
    - `description` (字串): 內容描述。
    - `assignee` (字串): 預期負責人。
- **列表顯示**:
    - `ID`: 系統序號。
    - `標題`: 支援完成狀態的視覺劃線。
    - `狀態`: 顯示為「已完成」(success) 或「未完成」(warning)。

## 操作行為 (Actions)
- **新增**: 清空表單並重新載入列表。
- **完成/拒絕/核准**: 根據任務當前所在的 BPM 階段，顯示不同的操作組合。
- **查看狀態**: 開啟彈窗顯示 BPMN 流程圖。
