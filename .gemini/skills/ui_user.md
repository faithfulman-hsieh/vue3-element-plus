# 界面定義：用戶管理 (UI User)

## 畫面描述
管理系統帳號與權限的後台介面。

## 欄位定義 (Fields)
- **新增用戶表單**:
    - `name` (名字): 字串，必填。
    - `email` (電子郵件): 字串，格式校驗，必填。
    - `password` (密碼): 字串，密碼遮罩。
    - `selectedRole` (角色): 下拉選單選擇 (ADMIN/USER)。
- **用戶列表**:
    - `index`: 序號。
    - `名字`: 用戶顯示名稱。
    - `電子郵件`: 帳號識別。

## 操作行為
- **列表載入**: 組件掛載後 (`onMounted`) 自動拉取用戶清單。
- **校驗**: 包含電子郵件格式正則校驗。
